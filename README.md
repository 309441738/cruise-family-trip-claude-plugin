# cruise-family-trip · Claude Code 插件

适合家庭出游的邮轮产品推荐，满足全家人的旅行需求和乐趣。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/cruise-family-trip-claude-plugin
/plugin install cruise-family-trip@cruise-family-trip-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **cruise-family-trip**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
